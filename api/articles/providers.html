Поставщики

Главная сущность внутренней части системы. Поставщики - компании, которые предоставляют услуги медицинский исследований.

В общем случае - поставщик является прослойкой между контрагентом и лабораторией. Контрагент получает заказы на исследование, забирает биоматериалы, и отправляет их поставщику, после чего работник поставщика распределяет эти заказы по лаборатория - в свою, или лаборатории аутсорс.

Provider

Базовая сущность поставщика, содержит базовую информацию о нем.

Свойства

  • name - название
  • provider2Cities - массив элементов типа Provider2City, который определяет список городов, где работает поставщик.

Provider2City

Сущность, которая определяет работу поставщика в конкретном городе.

Свойства

  • mainLabId - id лаборатории, которая является основной для поставщика в этом городе
  • providerId - id поставщика
  • cityId - id города
  • labs - массив сущностей типа Lab

Основная лаборатория

Собственная лаборатория поставщика, в которую отправляются все штативы кроме "Аутсорс".
